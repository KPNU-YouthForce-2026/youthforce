## 📌 Опис
**YouthForce** — це мобільний додаток, створений для об'єднання студентів та замовників (організацій). Платформа дозволяє студентам знаходити реальні проєкти для отримання практичного досвіду, а компаніям — делегувати завдання мотивованій молоді. 

**Ключовий функціонал:**
* **Для замовників:** Створення проєктів, встановлення дедлайнів, вимог до навичок та ліміту студентів; оцінювання роботи.
* **Для студентів:** Пошук проєктів, приєднання до ініціатив, формування портфоліо та підвищення власного рейтингу.
* Ромежування ролей на базі єдиної системи авторизації з автоматичним збереженням сесії.

## 🛠 Технології
* **Frontend:** Flutter / Dart
* **Backend & Auth:** Firebase Authentication
* **Database:** Cloud Firestore (NoSQL бази даних: колекції `students`, `employers`, `projects`)
* **Docs:** GitHub Wiki
